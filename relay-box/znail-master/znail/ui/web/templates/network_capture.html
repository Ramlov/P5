Network Capture

Network packages transfered to and from the SUT can be captured in PCAP format. Captured packages can then be analyzed using tools like Wireshark.

Download the capture.pcap file to begin. Package capturing starts automatically and continues until the download stops.

Captured packages can be streamed directly to Wireshark: curl {{ host_url }}capture.pcap | wireshark -k -i -
